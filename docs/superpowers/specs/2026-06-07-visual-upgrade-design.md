# Async Werewolf visual upgrade — design

Date: 2026-06-07

## Goal
Polish the site's visuals: bring the last two old-style role plates into the
field-guide engraving style, give the homepage a real editorial landing-page
treatment with a hero image, and refresh the Async Werewolf icon on
jacob-meacham.github.io's Lab page so it matches the new aesthetic.

## Scope

### 1. Regenerate old-style role plates
`werewolf` and `alpha-wolf` are the only roles still rendered in the old dark
painted-night style (full scene, moon). Every other role already uses the
hand-tinted copperplate "specimen plate" style. Their `ROLE_PROMPTS` entries are
already correct and subject-only, so a forced re-run suffices:

```
OPENAI_API_KEY=… npm run gen:roles -- --force --only werewolf,alpha-wolf
```

Edit prompts only if results regress.

### 2. New hero image (one image, two uses)
A bespoke **landscape** plate (`assets/img/hero.webp`, 1536×1024 = clean 3:2):
a snowbound 1720s Arctic Village establishing vignette under a faint cold moon,
in the same engraving style (cold midnight blue-green + candle-gold, plate-mark
border). Generated by `scripts/generate-hero.mjs`, which reuses the master-style
language but is landscape and allows a moon for this one establishing scene.

Because the image is exactly 3:2 it doubles as the Lab thumbnail — cohesive
across both sites.

### 3. Homepage → editorial landing page
Rework `index.md` + `_layouts/default.html` (homepage uses `hide_title`) and add
`_sass/_home.scss` (imported from `assets/css/style.scss`):

- **Hero band:** framed engraving + title "Async Werewolf", tagline, and CTA
  buttons → Rules / Roles / Arctic Village.
- **Lede** paragraph with a drop cap.
- **Victory Conditions** as two side-by-side callout cards (Village / Wolf).
- **Rules sections** keep their content but gain clearer hierarchy, section
  ornaments, and 3–4 inline decorative figures reusing existing role art
  (e.g. `seer`, `town-crier`, `werewolf`), captioned like field-guide plates.

### 4. Lab icon swap (jacob-meacham.github.io)
Replace pixel-art `img/werewolf-thumb.png` with the new hero engraving
(downscaled 3:2) and regenerate `img/werewolf-thumb.webp` via the repo's
existing `bin/thumbs.mjs` (sharp) pipeline. `lab/index.html` already references
those filenames.

## Out of scope (YAGNI)
Roles/Arctic-Village page redesign, new role art beyond the two, animation.

- `scripts/generate-hero.mjs` (new)
- regenerated `assets/img/{werewolf,alpha-wolf,hero}.webp`
- `index.md`, `_layouts/default.html`, `_sass/_home.scss` (new),
  `assets/css/style.scss`
- `jacob-meacham.github.io/img/werewolf-thumb.{png,webp}`
